Команды Git
Команда Описание
git init Создает новый репозиторий Git
git clone url Утилита командной строки Git для выбора существующего репозитория и создания его клона, т. е. копии.
git add file «Сохраняет» снимок текущего состояния проекта в истории коммитов
git add . Индексирует изменения
git commit -m "comment" Добавление комментария к изменениям
git commit -a -m "comment" Автоматически индексирует каждый уже отслеживаемый на момент коммита файл и добавляет комментарий
git push url Передаёт в удалённый репозиторий изменения, сделанные в локальном репозитории
git status Отображает состояние рабочего каталога и раздела проиндексированных файлов
git log Просмотра истории коммитов
git branch branch_name Добавление ветки
git checkout branch_name Перемещение на другую ветку
git checkout -b branch_name Создание новой ветки и сразу же переключение на неё
git remote Интерфейс для управления списком записей об удаленных подключениях, которые хранятся в файле /. git/config репозитория
